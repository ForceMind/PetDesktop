# Coco 桌宠 v1.8.0 群聊发布说明

## 推荐群公告版

大家好，Coco 桌宠 v1.8.0 更新啦！现在除了 Windows 和 macOS，还新增了可以直接打开的网页版。

Coco 支持透明置顶、拖动、滚轮缩放、鼠标注视、自动对白、5 套待机造型和 32 种完整逐帧动作。点击头、脸、左右手、身体或脚会触发不同效果；平时会自然站着，偶尔自己活动或表演。

这次重点修复了 Windows 滚轮缩放时的抖动闪烁。快速连续点击也不会再切断当前动画：Coco 会先完整演完，站稳后再播放排队的下一个动作，并给出对白提示。

网页版功能更多，可以自由点播全部动作，切换自动表演、待机动作、对白、换装和背景，还支持暂停、全屏、快捷键、设置记忆和安装为 PWA：
<https://forcemind.github.io/PetDesktop/>

桌面版下载：
<https://github.com/ForceMind/PetDesktop/releases/tag/v1.8.0>

Windows 下载 `CocoDesktopPet-Windows.exe`；macOS 下载 `CocoDesktopPet-macOS-universal.zip`。欢迎帮忙体验，有问题可以截图或录屏反馈！

## 精简转发版

Coco 桌宠 v1.8.0 发布啦！支持 Windows/macOS/Web，含 32 种完整动作、5 套待机造型、自动表演和双语对白。新版修复 Windows 滚轮闪烁，快速点击会排队，不再打断动作。网页版：<https://forcemind.github.io/PetDesktop/>；桌面下载：<https://github.com/ForceMind/PetDesktop/releases/tag/v1.8.0>

## English short post

Coco Desktop Pet v1.8.0 is available for Windows, macOS, and the web. This update fixes Windows wheel-resize flicker, adds a one-move interaction queue so animations are never interrupted, and introduces a richer installable web edition with all 32 actions, five idle outfits, controls, backgrounds, shortcuts, and saved settings. Web: <https://forcemind.github.io/PetDesktop/> · Downloads: <https://github.com/ForceMind/PetDesktop/releases/tag/v1.8.0>
