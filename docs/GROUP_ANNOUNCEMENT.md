# Coco 桌宠 v1.7.0 群聊发布说明

## 推荐群公告版

大家好，我做了一个 Coco 桌面宠物，Windows 和 macOS 都可以使用啦！

它是透明无边框窗口，可以拖动、滚轮缩放和始终置顶。点击头、脸、手、身体或脚会触发不同动画，目前共有 32 种完整逐帧动作，还有红围巾、蓝披风、圆眼镜和海军帽等换装。

这次的 v1.7.0 重点调整了待机节奏：Coco 平时会自然站着，不再不停点脚；偶尔会活动一下手脚，长时间不操作时还会自己随机表演，并弹出有趣对白。非中文系统默认纯英文，中文系统可以切换中文和 English。

下载地址：
<https://github.com/ForceMind/PetDesktop/releases/tag/v1.7.0>

Windows 下载 `CocoDesktopPet-Windows.exe`；macOS 下载 `CocoDesktopPet-macOS-universal.zip`。欢迎帮忙体验，有问题可以截图或录屏反馈！

## 精简转发版

Coco 桌宠 v1.7.0 发布啦！支持 Windows/macOS，透明置顶、拖动缩放、32 种点击动作、自动对白、鼠标注视和 4 套换装。新版平时会自然站着，偶尔自己活动或随机表演，不再一直点脚。下载：<https://github.com/ForceMind/PetDesktop/releases/tag/v1.7.0>
