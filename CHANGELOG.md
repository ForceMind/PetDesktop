# 变更记录

本文件记录面向用户的重要变更。版本发布包可在
[GitHub Releases](https://github.com/ForceMind/PetDesktop/releases) 获取。

## v1.4.2 - 2026-07-21

- 修复抬手时肩部与上臂之间出现白色月牙缝、看起来像手臂被分层拆开的错误。
- 新增四个从原始麻布纹理派生的运动关节衬层；衬层位于身体和四肢后方，按关节幅度淡入。
- 手臂衬层使用沿上臂方向的窄胶囊形状，并以手臂一半角度旋转，避免圆形补丁从身体后方露出。
- Windows 与 macOS 使用相同衬层资源和关节变换；新增白底连续动作预览与资源校验。

## v1.4.1 - 2026-07-21

- GitHub Windows Runner 改用离屏诊断模式验证启动、带换装动作渲染和正常退出，避免无交互桌面环境造成误报。
- Windows 与 macOS 产品代码和 `v1.4.0` 的连续骨骼动画保持一致。

## v1.4.0 - 2026-07-21

### 连续动画修复

- 生产渲染改回同一张原始 Coco 的五层原像素骨骼，完全停止加载轮廓和位置不一致的 256 张独立姿势图。
- Windows 与 macOS 每个画面都按连续曲线计算，32 个动作具有不同身体轨迹，至少 30 个动作带手、脚或头部关节运动。
- 所有关节曲线首尾归零，动作最后一帧与动态待机第一帧共享同一中立骨骼，不发生硬切闪烁。
- 恢复原图 `745:1205` 比例，并只使用横纵一致的缩放，修复角色拉伸。
- 收紧手脚遮罩、扩大关节重叠区，并限制极端旋转角度，减少肩部和腰部直切残片。

### 换装与验证

- 红围巾、蓝披风、圆眼镜和海军帽移入角色骨骼绘制栈，随跳跃、旋转、缩放和鼠标注视共同移动。
- Windows 构建只嵌入原图骨骼和换装资源；macOS App 同样不再复制独立动作帧。
- 连续性测试现在验证 32 条不同轨迹、中立首尾、原图比例、运行时单一骨骼和换装资源绑定。

## v1.3.1 - 2026-07-21

- Windows 构建改为先把 265 张生产帧打入单一内嵌归档，绕过 GitHub Actions 长工作目录下的
  Windows 命令行长度上限；最终交付仍是一个可双击运行的 EXE。

## v1.3.0 - 2026-07-21

### 逐帧动画重构

- 废弃“同一张角色图配合关节曲线和整体位移”的主动画方案。
- 新增 8 张动态待机画面，以及 32 个动作各 8 张独立动作帧，共 256 张动作画面。
- 每条动作时间线统一为 `base → 8 张动作帧 → base`；开始、结束和待机首帧引用同一个文件。
- 主关键帧之间新增真实补画的中间姿势，手臂、脚部、身体、表情和朝向均会随动作变化。
- 动作改为显示实际画面，不再交叉透明叠加大幅姿势，修复双手/双脚重影与闪烁。
- Windows 与 macOS 使用相同的帧命名、顺序和共同首尾帧状态机。

### 资源与测试

- 新增可复现的绿幕源表、帧处理脚本、`manifest.json` 和动作总览图。
- 自动校验 512 × 512 RGBA、8 张待机、256 张动作帧、32 条唯一时间线和共同 `base.png`。
- Windows 运行时测试逐一访问 256 张动作帧，并验证 32/32 动作从共同待机帧开始和结束。
- 应用改用方形透明画布等比绘制，避免不同姿势被挤压或拉伸。

## v1.2.0 - 2026-07-21

### 修复

- 恢复 `assets/coco.png` 作为唯一角色母版，移除旧版重新绘制的纸片人本体资源。
- 头和身体改为完整原图核心层，消除颈部矩形贴图、重复身体和明显拼接痕迹。
- 手臂与脚部改为从原图像素提取的同画布关节层，肩部和脚部接缝由身体覆盖区隐藏。
- Windows 与 macOS 实时渲染禁止横纵非等比缩放，修复角色拉伸和轮廓闪动。
- 渲染节奏稳定为 30 FPS，并减少 Windows 分层窗口不必要的尺寸更新。
- 校正鼠标左右跟随方向，并加入自动验证。

### 新增与调整

- 为原图关节层增加像素身份测试，防止角色被重新绘制或单独缩放。
- 红围巾、蓝披风、圆眼镜和海军帽重新定位到原始 Coco 身上。
- Windows ICO 与 macOS 图标源改为完整的原始 Coco 头像和蓝色羽冠。
- GitHub Actions 同时验证 Windows、macOS 源码、32 种动作、点击区域和原图身份。

## v1.1.0 - 2026-07-21

- 引入实时关节动画、四套换装和应用图标。
- 此版本使用了与原始 Coco 不一致的重绘纸片人部件，已由 `v1.2.0` 完全替换。

## v1.0.0 - 2026-07-21

- 首个 Windows 与 macOS 自动打包版本。
- 提供透明窗口、拖动、滚轮缩放、置顶菜单、中英文对白和 32 种交互入口。
